# RandomTreatmentForRandomData
A small, fast, mostly true random number generator routine, with both C/C++ native code and JavaScript implementations

Unlike typical methods, this random number generator gives you a different set of numbers every time it&rsquo;s used in a run without requiring a new &ldquo;seed&rdquo; value.  A JavaScript&reg; version appears here, along with a native code version for Windows&reg that&rsquo;s much faster than typical C standard library random number generators.  For a cross-platform native code version, refer to the <a href="https://github.com/kirkjkrauss/TypicallyTrueRandomNumberGeneratorForModernCPlusPlus">TypicallyTrueRandomNumberGeneratorForModernCPlusPlus</a> repository.

Random number generators serve many purposes, from data organization to cryptographic security, among a wide range of others.  A native code method for true random number generation can be useful for scalable data management.  A full stack method can help maximize cryptographic security.  Both of those use cases are discussed at <a href="https://developforperformance.com/RandomTreatmentForRandomData.html">https://developforperformance.com/RandomTreatmentForRandomData.html</a>.
